#  Iris Flower Classification

A Machine Learning project that classifies Iris flowers into three different species using Python and Scikit-learn.

---

## 📌 Project Overview

The Iris Flower Classification project predicts the species of an Iris flower based on four input features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

The model is trained using the famous Iris dataset and demonstrates the complete Machine Learning workflow from data preprocessing to prediction.

---

## 🎯 Objectives

* Learn Machine Learning fundamentals
* Perform data preprocessing
* Train a classification model
* Evaluate model performance
* Predict flower species accurately
